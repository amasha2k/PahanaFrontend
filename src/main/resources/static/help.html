How to Use Each Feature

Adding Customers

  1. Click "Add Customer" in the sidebar
  2. Fill in the customer details (name, address, telephone)
  3. The account number will be auto-generated
  4. Click "Add Customer" to save
Tip: Account numbers are automatically generated in the format CUST001, CUST002, etc.

Adding Items

  1. Click "Add Item" in the sidebar
  2. Enter item name, description, and price
  3. Set the initial stock level
  4. Click "Add Item" to save
Tip: You can update stock levels later from the "Manage Items" section.

Creating Orders

  1. Click "Add Order" in the sidebar
  2. Select a customer from the dropdown
  3. Choose items to add to the order
  4. Set quantities and prices for each item
  5. Apply discounts if needed
  6. Click "Create Order" to save
Important: Make sure the customer exists before creating an order.

Managing Orders

  1. Click "Manage Orders" in the sidebar
  2. View all orders in the table
  3. Use "Edit" to modify order details
  4. Use "Delete" to remove orders
  5. Use "Print" to generate bills
Success: All changes are automatically saved and reflected immediately.

Keyboard Shortcuts & Tips

  • Refresh: Press F5 to refresh the current page

  • Search: Use Ctrl+F to search within the current page
  • Back: Use browser back button or Alt+Left Arrow

  • Required Fields: Fields marked with * are mandatory
  • Auto-save: Some forms auto-save as you type
  • Validation: Forms validate input before submission
  • Cancel: Use the Cancel button to discard changes

Troubleshooting

Common Issues

Network Error

If you see "Network error" messages:

  • Check your internet connection
  • Refresh the page (F5)
  • Try again in a few moments
Form Not Submitting

If forms won't submit:

  • Check that all required fields are filled
  • Ensure valid data formats (email, phone numbers)
  • Try refreshing the page
Data Not Loading

If data doesn't appear:

  • Check the browser console for errors (F12)
  • Refresh the page
  • Contact support if the issue persists
Important: Always save your work before navigating away from forms. Data entered but not saved will be lost.
